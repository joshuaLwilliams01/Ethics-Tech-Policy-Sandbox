# Copy Audio File Instructions

The public folder exists at: `/Users/cristenwilliams/ethics-lab-p3/public`

## To copy your background-music.mp3 file:

**Option 1: Using Terminal (recommended)**
```bash
cp /Users/cristenwilliams/Downloads/background-music.mp3 /Users/cristenwilliams/ethics-lab-p3/public/background-music.mp3
```

**Option 2: Using Finder**
1. Open Finder
2. Navigate to: `/Users/cristenwilliams/Downloads`
3. Find `background-music.mp3`
4. Copy it (Cmd+C)
5. Navigate to: `/Users/cristenwilliams/ethics-lab-p3/public`
6. Paste it (Cmd+V)

**Option 3: If the filename is slightly different**
If your file has a different name (e.g., `Background Music.mp3` or `background-music.mp3`), rename it after copying:
```bash
cp "/Users/cristenwilliams/Downloads/your-actual-filename.mp3" /Users/cristenwilliams/ethics-lab-p3/public/background-music.mp3
```

After copying, the audio will automatically work with the toggle button!
